# Willkommens-App

Eine kleine Web-App, mit der du deinen Vornamen eingeben und eine persönliche Begrüßung erhalten kannst.

## Nutzung

1. Öffne `index.html` in einem Browser.
2. Gib deinen Vornamen in das Eingabefeld ein.
3. Klicke auf **Speichern**.
4. Die App zeigt dir "Hallo <Name>!" an.

Die App funktioniert vollständig ohne Backend und kann offline verwendet werden.
