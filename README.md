# Willkommens-App

Eine kleine Web-App, mit der du deinen Vornamen eingeben und eine persönliche Begrüßung erhalten kannst.

## Nutzung

1. Öffne `index.html` in einem Browser.
2. Gib deinen Vornamen in das Eingabefeld ein.
3. Klicke auf **Speichern**.
4. Nach der Begrüßung erscheint ein Button **Weiter** – klicke darauf, um das Animationsfenster mit dem roten Ball zu öffnen.
5. Im Fenster siehst du unten links einen Zähler, der jedes Mal erhöht wird, wenn der Ball eine Wand berührt.
6. Mit dem Button **Stop** unter dem Fenster schließt du die Animation wieder.

Die App funktioniert vollständig ohne Backend und kann offline verwendet werden.
